## Overview

AI Parameter Controls provides a clean, intuitive interface for users to adjust AI model parameters like temperature, top_p, and max_tokens. It includes:

- Visual sliders and inputs for each parameter
- Helpful tooltips explaining parameter effects
- Preset configurations for quick parameter sets
- Responsive vertical and horizontal layouts
- Reset controls to default values

## Parameter Options

Each parameter in the `parameters` array needs the following properties:

```tsx
interface AIParameter {
  name: string; // Parameter name (temperature, top_p, etc.)
  value: number; // Current value
  min: number; // Minimum allowed value
  max: number; // Maximum allowed value
  step: number; // Step increment for sliders
  description: string; // Helpful description for tooltips
}
```

## Preset Configuration

Presets allow users to quickly switch between parameter configurations:

```tsx
interface Preset {
  name: string; // Preset name (e.g., "Creative", "Precise")
  description: string; // Short description of the preset
  parameters: {
    // Parameter values for this preset
    [key: string]: number;
  };
}
```

The component includes default presets, but you can provide your own to match your application's needs.
