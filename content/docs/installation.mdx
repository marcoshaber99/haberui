## Prerequisites

haber-ui requires the following to be installed in your project:

- [shadcn/ui](https://ui.shadcn.com/)

## Steps

1. Set up shadcn/ui (if not already done)

<Tabs items={["pnpm", "npm", "yarn", "bun"]}>
  <Tab value="pnpm">```bash pnpm dlx shadcn@latest init ```</Tab>
  <Tab value="npm">```bash npx shadcn@latest init ```</Tab>
  <Tab value="yarn">```bash npx shadcn@latest init ```</Tab>
  <Tab value="bun">```bash bunx --bun shadcn@latest init ```</Tab>
</Tabs>

2. Install haber-ui components

```bash
npx shadcn@latest add "https://haberui.com/h/thinking-state.json"
```

Replace with the specific component you want to install.

Visit the [components page](/docs/components) to see the list of available components.
